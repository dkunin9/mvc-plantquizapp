# mvc-plantquizapp

simple iOS Quiz app for plants

### Running

Step 1:
```
git clone https://github.com/dkunin9/mvc-plantquizapp
```

Step 2: 
```
pod install
```

### Features

* Guess the plant by image
* Restart the quiz

### Pods used

* [SwiftyButton](https://github.com/TakeScoop/SwiftyButton)
